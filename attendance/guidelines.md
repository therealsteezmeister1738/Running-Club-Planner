# Attendance Tracking Guidelines

## Method

We use simple headcounts to track weekly attendance - no names or personal information collected.

## How to Log Attendance

1. Create a new file in `attendance/YYYY/` following the naming pattern: `YYYY-MM-DD-weekXX.md`
2. Use the template from `attendance/template.md`
3. Record the headcount for each scheduled run that week
4. Add optional notes (weather, special events, etc.)

## Privacy

- **No member names** are recorded
- **No personal data** is collected
- Headcounts only - aggregate numbers for tracking trends
- Files are public and searchable

## Example
```
Tuesday Evening Run: 12 attendees
Saturday Morning Long Run: 8 attendees
Weather: Light rain, 55°F
Notes: Good turnout despite weather!
```
